# Home renovation website
### See [result](https://spiffy-stroopwafel-e62006.netlify.app/)

##### Stack:
> Nuxt 3
> Typescript
> Pinia
> gsap
> aos
> swiper

## Clone

```bash
git clone https://github.com/interstellar-dev/home-renovation-website.git
````

## Install
```bash
# yarn
yarn install
```
### OR

```bash
# npm
npm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
```

### OR

```bash
npm run dev
```
